!> DO NOT install YSMenu on these carts. While some of them may appear similar to the timebomb clones from r4isdhc.com, none of these carts can use YSMenu as of now

These carts will use the Ace Wood 1.62 kernel. Ace Wood is based on an old version of R4 Wood, it’s not based on R4 Wood 1.62 as the name might suggest. It is missing some anti-piracy patches but nonetheless is good for game compatibility!

Installation is simple

  • Download Ace Wood 1.62 from here: flashcard-archive link
  • Extract it using your preferred file extractor
  • Copy the kernel files to the root of your micro SD card

Twilight menu ++

This cart supports twilight menu ++ You can install it by following these instructions: ds-homebrew wiki link

If you want to play some supported dsiware on a DS/Lite, this is an option to look into! The best part about Twilight menu ++ is that it is in active development, meaning that more dsiware will be supported in the future!

This also helps to get around some of the compatibility issues present in Ace Wood, twilight menu ++ will allow you to boot games from nds-bootstrap or from the cart's kernel (Ace Wood).

Twilight menu ++ also bundles in some emulators for other consoles, see the list here: ds-homebrew wiki link

Patching your games

There are two methods which can be used to patch your games, those are:

  • Applying AP patches using the nds ROM tool
  • Using a cheat database

Applying AP patches using the nds ROM tool

You can find NDS Scene tool from here: GBATemp Link.

To patch your rom, do the following:

  1. Open the ROM tool (Do not update if it prompts you to)
  2. Open the ROM in the tool by clicking the three dots, then select your rom. Or you can drag and drop your ROM into the tool.
  3. Click Export, make sure that ‘Apply AP Patch’ is ticked. If there is no AP patch available for your rom, this will not be an option.
  4. Export your ROM!

Using a cheat database

You can use cheats to enable AP patches without having to use the ROM tool. My go to cheat database is Deadskullzjr’s cheat database: GBATemp Link

To set it up on this cart, follow these steps:

  1. Download usrcheat.dat from the mega link in the GBATemp post
  2. Put it into the /_rpg/cheats/ folder of your micro SD card, replace/overwrite if prompted
  3. On the cart side, hover over the ROM you need to enable the patch for, press Y then X to open the cheats menu
  4. Select the AP patch by pressing A, then press X to save your selection
  5. Launch your game and enjoy!

If your cart is not working as expected, see the troubleshooting page here: Troubleshooting
