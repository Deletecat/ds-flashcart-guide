r4isdhc.com's Christmas cart!

This is my favourite cart, is it because it's a good cart? No. I just can't get over the idea of a Christmas R4 cart :p
Anyways, this cart is a little bit funky, here's how to set it up:

  • Download the Christmas kernel from here: flashcard-archive link
  • Extract it using your preferred file extractor
  • Copy the extracted kernel files to the root of your micro SD card!

You can stop here if you so wish, though keep in mind that game patches are outdated so newer games may not run. To counter this, we can install YSMenu. See this page for more details: R4 Clone YSMenu

If your cart is not working as expected, see the troubleshooting page here: Troubleshooting
