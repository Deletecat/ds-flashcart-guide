DSOne SDHC / DSOnei / DSOnei Mini - EvolutionOS

Unlike the DSOne, both the DSOne SDHC and DSOnei are not limited to 2GB micro SD cards! EvolutionOS comes with great game compatibility, as well as extra features like Real-Time Saving (RTS) and Game Slowdown.

The DSOne SDHC sometimes has issues with dead PSRAM. Especially the DSOne SDHC clones you can buy online nowadays. The next section will cover how to test this on your cart. I don't know if this issue applies to the DSOnei - I do know, however, that the self-test feature is not present on DSOnei/Mini carts.

Even if the PSRAM is dead, you can still use the cart with YSMenu, however you will miss out on EvolutionOS' special features.

How to test if your PSRAM is dead

  1. DSOne self-test
    • While holding L+R+UP, launch the cart.
    • This should take you to a screen saying Test Flash, press A twice to test the PSRAM then wait for the process to complete.
    • If you got the message saying 'psram ok!', you are free to carry on with this guide.
    • If you got the message saying 'psram error!', click here to be taken to the YSMenu guide
  2. Game test
    • Follow the instructions below to setup EvolutionOS
    • Put a .nds ROM onto your micro SD card, then launch it
    • If the game launches correctly, you can keep EvolutionOS
    • If games refuse to launch, delete all of the files on the micro SD card, then click here to be taken to the YSMenu guide

Installation is simple:

  • Download the kernel from here: flashcard-archive link
  • Extract it with your preferred file extractor
  • Copy the extracted kernel files to the root of your micro SD card

DSOnei Firmware Update

You can update the firmware of the DSOnei to launch it on unhacked DSi consoles on firmware version 1.4.5! Note that this tool will not work on a regular DSOne or DSOne SDHC. You can find the download here: flashcard-archive link

This does not work on the "DSOnei" carts on AliExpress. These are normal DSOne SDHC clones with a DSOnei sticker to make them more presentable.
