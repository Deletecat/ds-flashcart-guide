DSTT/i Clones - R4i.TP

These DSTTi clones boot from R4i.TP, they will use YSMenu.

Do not use the original TTMenu on these carts.

List of supported carts:

  • all carts from r4i-dshc.com
  • all carts from r4i-ndsill.com
  • all carts from r4i-yes.com
  • all carts from r4i-3ds.com

Installation is simple

  1. Download RetroGameFan's multicart update from here: GBATemp Link
  2. Extract it using your preferred file extractor
  3. Go into the DSTTi Clone YSMenu folder
  4. Copy all of the files in this folder to the root of your micro SD card
  5. Make a copy of TTMenu.dat, then rename it to R4i.TP

If your cart is not working as expected, see the troubleshooting page here: Troubleshooting
