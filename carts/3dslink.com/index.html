3dslink.com's Blue Cart

The 3dslink carts (red and blue) were gateway3ds clones. The blue cart let you play DS games. The red cart let you play 3DS games!
To be able to use the red cart, you had to use the blue cart to launch the exploit. This exploit works on firmware versions between 4.1.0 and 4.5.0, meaning you cannot use the red cart on newer firmware versions. The red cart is obsolete thanks to Custom Firmware (CFW), you can find a guide to install CFW here: 3ds.hacks.guide

This page takes you through setting up the blue cart instead.

Installation is simple

  • Download the 3dslink kernel from here: flashcard-archive link
  • Extract it using your preferred file extractor
  • Copy the kernel files to the root of your micro SD card

! > Older versions of the red cart firmware are known to brick 3DS consoles, you can thank gateway3ds for that...

If your cart is not working as expected, see the troubleshooting page here: Troubleshooting
