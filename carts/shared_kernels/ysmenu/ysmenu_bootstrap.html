YSMenu - M3Real clones (Bootstrap)

Carts listed on this page will use the M3Real YSMenu with Lifehackerhansol's flashcard bootstrap:

  • iTouchDS
  • R4(i) SDHC (r4rts.com)
  • R4 SDHC RTS black version (r4isdhc.com)

Installation is somewhat simple

  • Download RetroGameFan's multicart update from here: GBATemp link
  • Extract it using your preferred file extractor
  • Go into the M3Real_M3iZero YSMenu folder
  • Copy all of the files inside of this folder to the root of your micro SD card.

    • We're mostly done now, just a couple more steps!

  • Download Lifehackerhansol's YSMenu Bootstrap from here: GBATemp link
  • Extract it using your preferred file extractor
  • Copy the extracted bootstrap files to the root of your micro SD card!

If you want to learn more about the flashcard bootstrap, you can read about it here: GBATemp link

If your cart is not working as expected, see the troubleshooting page here: Troubleshooting
