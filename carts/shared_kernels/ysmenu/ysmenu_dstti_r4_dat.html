DSTT/i Clones - R4.dat

These DSTTi clones boot from R4.dat, they will use YSMenu.

Do not use the original TTMenu on these carts.

List of supported carts:

  • R4i SDHC Silver RTS Lite from r4isdhc.com - Without a year number and without the "NEW" label - using YSMenu on those carts will cause them to brick!
  • R4iTT from r4itt.net - Must have a screw on the back of the cart, if it doesn't this screw, the cart will brick if you try to use YSMenu!
  • all carts from r4iiisdhc.com

Installation is simple

  1. Download RetroGameFan's multicart update from here: GBATemp Link
  2. Extract it using your preferred file extractor
  3. Go into the DSTTi Clone YSMenu folder
  4. Copy all of the files in this folder to the root of your micro SD card
  5. Make a copy of TTMenu.dat, then rename it to R4.dat

If your cart is not working as expected, see the troubleshooting page here: Troubleshooting
