DSOne/i YSMenu

Here are the list of carts that are supported:

  • DSOne SDHC
  • DSOnei SDHC
  • DSOnei Mini

!> The original DSOne is not supported by YSMenu.

Installation (Main kernel)

  1. Download Retrogamefan’s multicart update from gbatemp: GBATemp Link
  2. Extract it using your preferred file extractor
  3. Go into the DSONEi SDHC YSMenu folder
  4. Copy the following necessary files to your micro SD card:
    • Games folder
    • TTMenu Folder
    • SCFW.SC
    • TTMenu.dat
    • You can copy over MSFORSC.nds if you want - though this is not required to get the cart working

Installation (Dual-boot)

  1. Download EvolutionOS from here: flashcard-archive link
  2. Extract it using your preferred file extractor
  3. Copy the extracted files over to your micro SD card
  4. Download Retrogamefan’s multicart update from gbatemp: GBATemp Link
  5. Extract it using your preferred file extractor
  6. Go into the DSONEi SDHC YSMenu folder
  7. Copy the following necessary files to your micro SD card:
    • Games folder
    • TTMenu Folder
    • TTMenu.dat
    • MSFORSC.nds

To boot YSMenu, first boot into EvolutionOS, then launch MSFORSC.nds to boot into YSMenu!

If your cart is not working as expected, see the troubleshooting page here: Troubleshooting
