DSTT/i and it’s clones

These are carts which use the same kernel and boot file as the DSTT/i

Do not use the original TTMenu on these carts unless you are certain that it isn't a DSTT clone.

List of DSTTi/clones

  • ndstt.com / ndstt.net - DSTTi
  • ndstti.cn - DSTTi Gold
  • dstt-adv.com - DSTT-Advance
  • r4top.com - R4Top Revolution
  • r4ll-net.com - R4iSDHC v1.41 and v1.4.2 (NOT R4i-SDHC.COM)
  • r4ill-net.com - v1.45 (possibly others)

YSMenu

Setting up YSMenu on these carts is as simple as:

  1. Downloading Retrogamefan’s multicart update from here: https://gbatemp.net/download/retrogamefan-multi-cart-update.35737/download
  2. Extract it using your preferred file extractor
  3. Go into the DSTTi Clone YSMenu folder
  4. Copy all of the files in this folder to the root of your micro SD card

If your cart is not working as expected, see the troubleshooting page here: Troubleshooting
