YSMenu - DSTT/i Clones - Unknown boot files

If you have been linked to this page, you have a DSTT/i clone - but we do not know which file it will boot from. Follow the instructions down below to figure out which boot file your cart may use. Once you have figured that out, please open an issue on GitHub so that we can link to the correct page in the future!

Do not use the original TTMenu on these carts.

Installation is easy, but slightly time consuming

  1. Download RetroGameFan's multicart update from here: GBATemp Link
  2. Extract it using your preferred file extractor
  3. Go into the DSTTi Clone YSMenu folder
  4. Copy all of the files in this folder to the root of your micro SD card
    If it works, great! if not, then continue on;
  5. Make 6 copies of TTMenu.dat and rename them to:
    • R4i.TP
    • iLL.iL
    • config
    • 3DSCARD.dat
    • R4i.3DS
    • R4i.PK
  6. if it boots, delete the files one by one until the cart stops booting into YSMenu. That will be the correct boot file.
  7. Delete the other boot files (keep TTMenu.dat), then create a copy of TTMenu.dat with the name of the correct boot file.

If the cart still didn't boot, open up a GitHub issue.

If your cart is not working as expected, see the troubleshooting page here: Troubleshooting
